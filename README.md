# AI Student Management System

## 📌 Project Overview

AI Student Management System is a web-based application designed to streamline academic management by providing role-based access for Faculty and Students.

The system enables efficient management of student records, attendance tracking, marks management, performance prediction, and AI-generated reports through an interactive and user-friendly interface.

---

## 🚀 Key Features

### 👨‍🏫 Faculty Module

* Faculty Registration and Login
* Student Management

  * Add Students
  * Update Student Details
  * Delete Students
* Attendance Management
* Marks Management
* Performance Prediction
* AI Report Generation
* Faculty Profile Management

### 👨‍🎓 Student Module

* Student Login
* View Attendance Records
* View Marks
* View Performance Prediction
* View AI Reports
* Account Settings
* Password Management

### 🤖 AI Features

* Student Performance Prediction
* AI-Based Academic Insights
* Automated Performance Reports

## 📊 Modules Implemented

### Student Management

Manages complete student information including:

* Student ID
* Name
* Email
* Class
* Section
* Phone Number

### Attendance Management

Faculty can:

* Mark attendance
* Update attendance records
* Track attendance percentage

Students can:

* View attendance percentage
* View attendance history

### Marks Management

Faculty can:

* Add subject marks
* Update marks

Students can:

* View marks
* View average marks

### Performance Prediction

The system predicts student performance using:

* Attendance Percentage
* Average Marks

Prediction Categories:

* Excellent
* Good
* Needs Improvement

### AI Report Generation

Generates detailed student reports including:

* Attendance Analysis
* Academic Performance Analysis
* Performance Prediction
* Improvement Suggestions
